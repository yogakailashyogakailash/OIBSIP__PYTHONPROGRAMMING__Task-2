🧮 BMI Calculator (Python Tkinter)

A simple GUI-based Body Mass Index (BMI) Calculator built using Python and Tkinter.
This application allows users to enter their weight and height and calculates their BMI along with the health category.

🚀 Features

🖥️ Simple graphical user interface (GUI)

⚖️ Calculates Body Mass Index (BMI)

📊 Displays BMI category (Underweight, Normal, Overweight, Obese)

❗ Error handling for invalid inputs

🎯 Easy and beginner-friendly Python project

🛠️ Technologies Used

Python

Tkinter (GUI library)

📦 Requirements

Python 3.x installed on your system.

Tkinter usually comes pre-installed with Python.
To check:

python -m tkinter
▶️ How to Run

Clone the repository

git clone https://github.com/your-username/bmi-calculator.git

Go to the project folder

cd bmi-calculator

Run the program

python bmi_calculator.py
📋 How It Works

Enter Weight in kilograms (kg)

Enter Height in meters (m)

Click Calculate BMI

The program displays:

BMI value

BMI category

📊 BMI Categories
BMI Range	Category
Less than 18.5	Underweight
18.5 – 24.9	Normal
25 – 29.9	Overweight
30 and above	Obese
